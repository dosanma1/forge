# Tab

#### Navigation component for organizing content into sections.

<Canvas sourceState="hidden" of={TabStories.PillHorizontal} />

## Features

- ✅ Keyboard navigation
- ✅ Accessible tabs pattern
- ✅ Built with ng-primitives

## Import

```typescript
import { MmcTab, MmcTabList, MmcTabPanel } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<ng-container ngpTabs>
	<mmc-tab-list ngpTabList>
		<button ngpTab>Tab 1</button>
		<button ngpTab>Tab 2</button>
	</mmc-tab-list>
	<mmc-tab-panel ngpTabPanel>Content 1</mmc-tab-panel>
	<mmc-tab-panel ngpTabPanel>Content 2</mmc-tab-panel>
</ng-container>
```
