# Font Manager

The Font Manager component allows users to adjust the global font size of the application using percentage-based scaling. Changes are persisted to localStorage and proportionally affect all text sizes.

## Features

- **5 Font Size Options**: Choose from Smaller, Small, Default, Large, and Larger
- **Percentage-Based Scaling**: Proportionally scales all text relative to the base font size
- **Persistent Settings**: Font size preferences are saved to localStorage
- **Global Application**: Changes affect the entire application's text proportionally
- **Accessible**: Helps users with visual preferences customize text readability
