# Table

#### Complex data tables with sorting, filtering, and pagination.

## Features

- ✅ Sortable columns
- ✅ Filterable data
- ✅ Pagination
- ✅ Row selection
- ✅ Custom cell templates

## Import

```typescript
import {
	MmcTable,
	MmcTableHeader,
```

## Examples

### Basic Usage

```html
<mmc-table [data]="tableData" [columns]="columns"> </mmc-table>
```

### With Sorting

```html
<mmc-table
	[data]="data"
	[columns]="columns"
	[sortable]="true"
	(sortChange)="onSort($event)"
>
</mmc-table>
```
