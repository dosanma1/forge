# Confirmation Dialog

#### Modal dialog for user confirmations.

<Canvas sourceState="hidden" of={ConfirmationDialogStories.Default} />

## Features

- ✅ Modal overlay
- ✅ Focus trap
- ✅ Keyboard dismissible
- ✅ Customizable actions

## Import

```typescript
import { MmcConfirmationDialog } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<mmc-confirmation-dialog
	[(open)]="isOpen"
	title="Confirm Action"
	message="Are you sure you want to delete this item?"
	(confirm)="onConfirm()"
	(cancel)="onCancel()"
>
</mmc-confirmation-dialog>
```
