# Textarea

#### A multi-line text input field for longer text content.

<Canvas sourceState="hidden" of={TextareaStories.Default} />

## Features

- ✅ Multi-line text input
- ✅ Resizable or fixed height
- ✅ Disabled state
- ✅ Works with Angular forms
- ✅ Customizable with Tailwind classes

## Import

Get started with importing the directive:

```typescript
import { MmcTextarea } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<textarea
	mmcTextarea
	placeholder="Type something..."
	class="min-h-20"
></textarea>
```

### With Label

```html
<div class="grid gap-2">
	<label mmcLabel for="message">Your message</label>
	<textarea
		mmcTextarea
		id="message"
		placeholder="Leave a comment..."
		class="min-h-24"
	></textarea>
	<p class="text-sm text-muted-foreground">
		Your message will be copied to the support team.
	</p>
</div>
```

### With Reactive Forms

```typescript
import { FormControl } from '@angular/forms';

export class MyComponent {
	message = new FormControl('');
}
```

```html
<textarea mmcTextarea [formControl]="message" class="min-h-32"></textarea>
```

### Resizable

```html
<textarea
	mmcTextarea
	placeholder="This textarea can be resized"
	class="min-h-32 resize"
></textarea>
```

### Fixed Height

```html
<textarea
	mmcTextarea
	placeholder="This textarea has a fixed height"
	class="h-32 resize-none"
></textarea>
```

## Accessibility

- Supports all native textarea attributes
- Works with labels using `id` and `for` attributes
- Respects ARIA attributes
- Keyboard accessible
