# Select

#### A dropdown component for selecting from a list of options.

<Canvas sourceState="hidden" of={SelectStories.Default} />

## Features

- ✅ Keyboard navigation
- ✅ Single and multi-select
- ✅ Works with Angular forms
- ✅ Customizable options
- ✅ Search/filter capability

## Import

```typescript
import { SelectComponent, OptionComponent } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<mmc-select [(ngModel)]="selected" placeholder="Choose an option">
	<mmc-option value="1">Option 1</mmc-option>
	<mmc-option value="2">Option 2</mmc-option>
	<mmc-option value="3">Option 3</mmc-option>
</mmc-select>
```

### With Reactive Forms

```typescript
selectedOption = new FormControl('');
```

```html
<mmc-select [formControl]="selectedOption">
	<mmc-option value="apple">Apple</mmc-option>
	<mmc-option value="banana">Banana</mmc-option>
</mmc-select>
```
