# Listbox

#### A keyboard-navigable list of selectable options.

<Canvas sourceState="hidden" of={ListboxStories.Default} />

## Features

- ✅ Keyboard navigation
- ✅ Single and multi-select
- ✅ Built with ng-primitives

## Import

```typescript
import { MmcListbox, MmcListboxOption } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<mmc-listbox [(ngModel)]="selected">
	<mmc-listbox-option value="1">Option 1</mmc-listbox-option>
	<mmc-listbox-option value="2">Option 2</mmc-listbox-option>
</mmc-listbox>
```
