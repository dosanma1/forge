# Tooltip

#### Contextual information that appears on hover or focus.

<Canvas sourceState="hidden" of={TooltipStories.Default} />

## Features

- ✅ Appears on hover and focus
- ✅ Multiple positions (top, bottom, left, right)
- ✅ Keyboard accessible
- ✅ Built with ng-primitives

## Import

```typescript
import { MmcTooltip, MmcTooltipContent } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<button mmcButton [tooltip]="'Helpful information'">Hover me</button>
```

### With Tooltip Component

```html
<ng-template #tooltipContent>
	<mmc-tooltip-content>This is a tooltip</mmc-tooltip-content>
</ng-template>

<button mmcButton [ngpTooltip]="tooltipContent">Hover me</button>
```

## Accessibility

- Keyboard navigable
- Screen reader friendly
- Follows WAI-ARIA tooltip pattern
