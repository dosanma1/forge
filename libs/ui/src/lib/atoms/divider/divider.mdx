# Divider

#### A visual separator for content sections.

<Canvas sourceState="hidden" of={DividerStories.Default} />

## Features

- ✅ Horizontal and vertical orientations
- ✅ Customizable with Tailwind classes
- ✅ Semantic HTML element

## Import

Get started with importing the component:

```typescript
import { MmcDivider } from '@mmc/ui';
```

## Examples

### Horizontal Divider

```html
<p>Content above</p>
<mmc-divider />
<p>Content below</p>
```

### Vertical Divider

```html
<div class="flex">
	<p>Left content</p>
	<mmc-divider class="ml-4" orientation="vertical" />
	<p>Right content</p>
</div>
```

### Custom Spacing

```html
<mmc-divider class="my-8" />
```
