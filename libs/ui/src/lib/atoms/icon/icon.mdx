# Icon

The Icon component provides a consistent way to display Lucide icons throughout the application with customizable size and color properties.

## Features

- **Lucide Icons**: Access to the entire Lucide icon library
- **Multiple Sizes**: Predefined size variants (xs, sm, md, lg, xl)
- **Custom Sizing**: Support for custom width and height
- **Color Customization**: Easily apply custom colors via CSS classes
- **Accessible**: Proper aria attributes for screen readers

## Sizes

| Size | Dimension | Usage                              |
| ---- | --------- | ---------------------------------- |
| xs   | 12px      | Inline text icons                  |
| sm   | 16px      | Small UI elements                  |
| md   | 20px      | Default size for most cases        |
| lg   | 24px      | Section headers, prominent actions |
| xl   | 32px      | Hero sections, large displays      |

## Usage

```typescript
import { MmcIcon } from '@forge/ui';

@Component({
  template: `
    <mmc-icon name="lucideHome" size="md"></mmc-icon>
    <mmc-icon name="lucideSettings" [width]="24" [height]="24"></mmc-icon>
    <mmc-icon name="lucideUser" size="lg" class="text-primary"></mmc-icon>
  `
})
```
