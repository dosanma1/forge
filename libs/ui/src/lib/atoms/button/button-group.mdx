# Button Group

#### A container for grouping related buttons together.

<Canvas sourceState="hidden" of={ButtonGroupStories.Default} />

## Features

- ✅ Groups buttons visually
- ✅ Consistent spacing
- ✅ Horizontal and vertical layouts

## Import

```typescript
import { MmcButtonGroup } from '@mmc/ui';
```

## Examples

### Basic Usage

```html
<mmc-button-group>
	<button mmcButton>Left</button>
	<button mmcButton>Middle</button>
	<button mmcButton>Right</button>
</mmc-button-group>
```

### Vertical

```html
<mmc-button-group class="flex-col">
	<button mmcButton>Top</button>
	<button mmcButton>Middle</button>
	<button mmcButton>Bottom</button>
</mmc-button-group>
```
