# Slider

A slider component that allows users to select a value from a range by dragging a thumb along a track.

## Features

- **Value Range**: Configure minimum, maximum, and step values
- **Accessible**: Full keyboard navigation support
- **Form Integration**: Works seamlessly with Angular forms via `ControlValueAccessor`
- **Customizable**: Easily styled with Tailwind CSS classes

## API

### Inputs

- `min` - Minimum value (default: 0)
- `max` - Maximum value (default: 100)
- `step` - Step increment (default: 1)
- `disabled` - Whether the slider is disabled
- `orientation` - Slider orientation: `horizontal` or `vertical`
- `class` - Additional CSS classes

### Outputs

- `valueChange` - Emitted when the value changes

## Keyboard Navigation

- **Arrow Left/Down**: Decrease value by one step
- **Arrow Right/Up**: Increase value by one step
- **Shift + Arrow**: Change value by 10 steps
- **Home**: Jump to minimum value
- **End**: Jump to maximum value

## Form Integration

The slider component implements `ControlValueAccessor` and can be used with Angular forms:

```typescript
import { FormControl } from '@angular/forms';

volumeControl = new FormControl(50);
```

```html
<mmc-slider [formControl]="volumeControl" [min]="0" [max]="100"></mmc-slider>
```
